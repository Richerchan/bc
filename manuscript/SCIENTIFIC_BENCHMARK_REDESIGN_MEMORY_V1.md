# Scientific Benchmark Redesign Memory V1

## What this round established

This manuscript should no longer rely on the older 98-case internal-heavy benchmark as its primary evidence base. The current default evidence package is the redesigned three-track benchmark:

- Track A: 30 ablation cases across six reliability families and five scientific themes
- Track B: 36 science-facing cases across constants/units, condition-sensitive properties, and abstention-boundary questions
- Track C: 20 longitudinal cases organized as 10 two-step sequences

The benchmark now executes 318 runs in total.

## Core writing rules fixed by this round

- The paper must describe the benchmark as a unified case-schema system, not as an ad hoc collection of examples.
- Track B is the main science-facing evidence, not a side benchmark.
- Track A should be written as a control-structure isolation result.
- Track C should be written as bounded sequence evidence, not as proof of mature long-horizon memory.
- Results must distinguish:
  - governance gains
  - answer-yield weaknesses
  - memory carryover limits

## Core experimental lessons fixed by this round

- Typed mirror control contributes the largest aggregate gain in Track A.
- Memory improves targeted families such as project-local precedence and some correction-sensitive cases, but does not automatically improve every family.
- In Track B, direct answering can appear numerically strong while remaining scientifically unreliable because it drops source conditions and provenance.
- In Track B, abstention and provenance governance are current strengths.
- In Track B, condition-sensitive answering remains the main weakness: the current system detects condition dependence more reliably than it produces high-yield conditioned answers.
- In Track C, correction lineage helps, but recurrence is not fully removed and scope isolation remains weak.

## Persistent interpretation boundary

- "bounded scientific reasoning" is still the correct title-level framing.
- The paper should not claim broad scientific superiority.
- The strongest defensible claim is:
  explicit state + typed critique + correction-lineage memory improve auditable scientific claim governance within the current local benchmark.
